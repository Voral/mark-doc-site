# Веб-представление для документации хранящийся в Markdown файлах

Небольшой проект для отображения документации хранящейся в Markdown файлах.

## Описание

Это Laravel-приложение, которое предоставляет веб-интерфейс для просмотра документации, организованной в виде
Markdown-файлов. Приложение автоматически сканирует указанную директорию и формирует навигационное меню на основе
структуры папок и файлов.

## Особенности

- Автоматическое создание навигации по структуре папок
- Поддержка вложенных директорий
- Отображение содержимого Markdown-файлов
- Поддержка заголовков в Markdown-файлах
- Адаптивный дизайн
- Возможность кастомизации путей к документации

## Требования:

- PHP 8.2 и выше
- Composer
- Laravel 12.0 и выше

## Установка

1. Клонируйте репозиторий
2. Установите зависимости через Composer:
    ```bash
    composer install
    ```
3. Скопируйте .env.example в .env и настройте параметры:
    ```bash
    cp .env.example .env
    ```
4. Сгенерируйте ключ приложения:
    ```bash
    php artisan key:generate
    ```
5. Настройте путь к документации в .env:
   ```
   MARK_DOC_PATH=../path/to/your/markdown/files
   ```
6. Соберите assets:
   ```bash 
    pnpm install && pnpm run build
   ```
## Настройка

### Путь к документации

Путь к документации должен быть указан в .env в переменной MARK_DOC_PATH. Путь может быть абсолютным или относительным к корню проекта.

## Структура документации

Структура документации формируется на основе структуры папок и файлов:

```
docs/
├── intro.md
├── installation/
│   ├── requirements.md
│   └── setup.md
└── api/
    ├── endpoints/
    │   ├── users.md
    │   └── posts.md
    └── authentication.md
```

Заголовок первого уровня из файла будет отображаться как заголовок страницы.
